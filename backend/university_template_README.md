# قالب استيراد الجامعات

- مكان الملف: `backend/university_template.csv`
- يمكنك فتح هذا الملف في Microsoft Excel أو LibreOffice Calc ثم حفظه كـ `.xlsx`.

## الأعمدة المدعومة

| الاسم الإنجليزي | الاسم العربي | الإلزامية | الوصف |
|---|---|---|---|
| `name` أو `Name` | `اسم` | **مطلوب** | اسم الجامعة |
| `website` أو `Website` | `موقع` | اختياري | رابط كامل مثل `https://...` |
| `country` أو `Country` | `دولة` | اختياري | استخدم `Turkey` أو `Cyprus` |
| `description` أو `Description` | `وصف` | اختياري | نبذة عن الجامعة |
| `logo` أو `Logo` | `شعار` | اختياري | **رابط URL** لصورة الشعار (يجب أن يبدأ بـ `http`) |

## ملاحظات مهمة حول عمود الشعار (logo)

- يجب أن يكون القيمة **رابط URL مباشر** للصورة يبدأ بـ `https://` أو `http://`
- لا يمكن رفع صورة مباشرة من الجهاز عبر الاكسل — يجب استخدام رابط مباشر
- مثال على رابط صحيح: `https://example.com/logo.png`
- إذا تركت الخلية فارغة، ستُعرض الجامعة بالحروف الأولى من اسمها بدلاً من الشعار
- الروابط التي لا تبدأ بـ `http` يتم تجاهلها تلقائياً

## قواعد عامة
- أي صف بدون قيمة في عمود `name` سيتم تجاهله.
- الأسماء المكررة تُتجاوز لتفادي إدخال نفس الجامعة مرتين.

## مثال

```
name,website,country,description,logo
Istanbul University,https://www.istanbul.edu.tr,Turkey,"جامعة عريقة في إسطنبول",https://example.com/istanbul-logo.png
جامعة قبرص الدولية,https://www.ciup.edu.cy,Cyprus,"جامعة خاصة في شمال قبرص",
```
