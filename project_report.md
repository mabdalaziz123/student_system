# تقرير مشروع UniPortal / Nokta Code

## 1. نظرة عامة على المشروع (Project Overview)
مشروع **UniPortal** (المعروف أيضاً بـ **Nokta Code**) هو نظام متكامل لإدارة تسجيل الطلاب والجامعات. يهدف النظام إلى تسهيل عملية الربط بين الطلاب الراغبين في الدراسة والجامعات المتاحة، مع توفير واجهة إدارة (Dashboard) قوية للمسؤولين لمتابعة الطلبات وتدقيق البيانات.

## 2. الهوية البصرية والعلامة التجارية (Visual Identity)
تم تغيير العلامة التجارية للمشروع مؤخراً من "نوكتا كودز" (NOKTA CODES) إلى **"نقطة كود" (Nokta Code)**.
- **الألوان الأساسية:** يعتمد النظام على واجهة عصرية تدعم الوضع الليلي (Dark Mode) مع استخدام مكتبة `lucide-react` للأيقونات.
- **التجربة البصرية:** التركيز على البساطة وسهولة الوصول للمعلومات عبر أقسام منظمة.

## 3. البنية التقنية (Tech Stack)
يعتمد النظام على بنية Full-Stack حديثة:
- **الواجهة الأمامية (Frontend):** 
  - **React 19** مع **Vite** كأداة بناء.
  - **TypeScript** لضمان جودة الكود.
  - استخدام **Vanilla CSS** للتنسيق.
- **الخلفية (Backend):** 
  - **Flask (Python)** كإطار عمل للـ API.
  - **Flask-SQLAlchemy** للتعامل مع قاعدة البيانات.
  - **Flask-CORS** لتمكين الاتصال بين الواجهة الأمامية والخلفية.
- **قاعدة البيانات (Database):** 
  - **PostgreSQL** لتخزين البيانات العلائقية (Relational Data).

## 4. نماذج البيانات الأساسية (Data Models)
يتكون النظام من عدة كيانات مترابطة:
- **المستخدمون (Users):** إدارة الحسابات بصلاحيات مختلفة (مثل ADMIN).
- **الطلاب (Students):** تخزين بيانات الطلاب الشخصية (الاسم، جواز السفر، الجنسية، إلخ).
- **الجامعات (Universities):** إدارة بيانات الجامعات ومواقعها الإلكترونية ووصفها.
- **البرامج الدراسية (Programs):** التخصصات المتاحة في كل جامعة مع تفاصيل الأقساط واللغة.
- **الطلبات (Applications):** الربط بين الطلاب والبرامج الدراسية ومتابعة حالة الطلب.

## 5. المميزات الرئيسية (Core Features)
- **لوحة التحكم (Dashboard):** عرض إحصائيات سريعة عن عدد الطلاب والطلبات والبرامج.
- **إدارة الجامعات والبرامج:** إمكانية إضافة وتعديل وحذف البيانات بسهولة.
- **تقديم الطلبات:** نظام لرفع وثائق الطلاب وربطها بالتخصصات المختارة.
- **نظام المصادقة (Authentication):** تسجيل دخول آمن وإدارة مستخدمين.
- **دعم الذكاء الاصطناعي:** تكامل مع **Google Gemini AI** لتوفير ميزات ذكية داخل النظام.

## 6. تعليمات التشغيل (Setup Instructions)
### تشغيل الخلفية (Backend):
1. تثبيت المتطلبات: `pip install -r backend/requirements.txt`
2. ضبط ملف `.env` لبيانات قاعدة البيانات.
3. تشغيل الخادم: `python backend/run.py`

### تشغيل الواجهة الأمامية (Frontend):
1. تثبيت الحزم: `npm install`
2. ضبط مفتاح Gemini API في ملف `.env.local`.
3. تشغيل المشروع: `npm run dev`
